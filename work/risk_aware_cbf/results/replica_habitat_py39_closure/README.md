# Replica Habitat-Sim Python 3.9 Closure

G0.5D-v2 closes the isolated official Python 3.9 renderer environment and
single-frame scene smoke gates. Formal Replica preprocessing is intentionally
not run: the inherited baseline contains no frozen Replica camera trajectory,
frame-selection seed, or train/eval split contract. No RGB, depth, mesh,
navmesh, or Conda artifact is committed here.
