# Formal TUM Splatfacto Training Execution V1

This directory records the one authorized execution of the frozen TUM FR1 Room
Splatfacto protocol. It contains only compact provenance, validation, and report
artifacts. Checkpoints, complete logs, TensorBoard files, and renders remain on
the authoritative server.

The training protocol is immutable at
`../tum_splatfacto_training_protocol_v1/`. No SAFER loader, ellipsoid query,
navigation, baseline, or G1 evaluation is part of this task.
